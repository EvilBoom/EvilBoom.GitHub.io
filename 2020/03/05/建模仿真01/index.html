1.绪论

  1. 仿真科学概述
    • 包饺子问题
      • 考虑：3要素
      • 背景是否明确
      • 目标是否明确，是否可以量化
      • 假设(约束)是否明确
        1. 饺子皮是圆的
        2. 皮的厚度是固定的
        3. 同样面积的饺子皮包的陷是固定的
        4. 给定：面粉的数量一定
    • 饺子求解思路：
      • 面粉用体积v来表示
      • 能报多少馅用表面积s表示
      • 在给定v下，想要报完s，可以包n个
      • 每个饺子的体积 v1 = s/n => 体积转化表面积问题
      • so v可以转化成表面积Sv
    • 重点：拿到一个问题首先考虑什么，怎么转化成数学问题来推导证明
    • 一大类问题：综合评价问题
    • hw2
      • 如果通过样本来验证一个分布是哪种分布
      • 分布函数F怎么确定
      • 怎么检验结论是对的
      • 假设检验，参数估计
    • 三种思维： 理论思维，实验思维，仿真思维
    • hw3 对话的逻辑谬误
