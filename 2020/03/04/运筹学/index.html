运筹学

  1. 怎么理解运筹学--为什么叫 。。 学
    • 有理论基础，并且这套理论应该是很大程度上被证明了的
    • 相对独立
    • 如果不能被成为学，可能是这套方法并没有办法被证明时最优的
  2. 线性规划
    1. 概述
      • 门要关上，有三个约束点
      • 表达式会产生很多解，我们就需要从一堆解中找到使目标函数最优的解
      • 寻优的过程
    2. x+200 = 500 x最优为300
      • x1 + x2 = 500
        • x1 无限制条件
        • x2 有限制条件
        • target 2x1+x2
        • x1 + x2 < 1000
