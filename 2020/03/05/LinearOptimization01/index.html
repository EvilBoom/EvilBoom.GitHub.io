What is Linear Programming

  1. Programming -- 这里其实就是 Optimization 的意思，使program work 就是 programming ，把 program 做好，就是Optimization

    • 优化通俗来说可以说是在一定条件下尽可能满足一些需求

  2. Linear 线性就是限制的条件是以线性的形式表达

  3. LP 就是优化一个线性的目标函数，其中的决策变量服从一个由线性约束组成的集合。

    • Example
      • Minimize(目标函数) \(x^2\)

\[\frac{\partial u}{\partial t} = h^2 \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} + \frac{\partial^2 u}{\partial z^2}\right)\]
